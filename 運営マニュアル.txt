=====================================================
 YeG Tourney Bot 運営マニュアル
=====================================================
最終更新: 2026-06-11

このファイルは大会運営者向けの「いつでも見れる」手引きです。
困ったらまずここを見てください。


-----------------------------------------------------
■ 1. Botの起動方法
-----------------------------------------------------
PowerShell（またはターミナル）で以下を実行:

    cd C:\Users\25B1029\yeg-tourney-bot
    python bot.py

「ログイン成功: ...」と表示されれば起動完了。
このウィンドウは大会中ずっと開いたままにしておくこと（閉じるとBotが止まる）。

止めたいとき: そのウィンドウで Ctrl + C


-----------------------------------------------------
■ 2. 大会当日の進行フロー
-----------------------------------------------------
上から順にコマンドを打つだけで進みます。
迷ったら /tourney-status を見れば「次にやる操作」が必ず表示されます。

  事前準備
    /tourney-create 第1回タイピング大会   … 大会を作成
    /tourney-open                          … エントリー受付開始（ボタン付き告知が出る）
                                              → 参加者はボタンから登録

  受付終了
    /tourney-close                         … 受付締切（参加者一覧が出る）

  対戦開始
    /tourney-seed ランダム                 … 対戦表を自動生成（または「申込順」）

  試合ごとに繰り返し
    /tourney-next                          … 次の対戦カードを表示・対戦者にメンション
    （試合）
    /tourney-result R1M1 プレイヤー1 100-80 … 結果入力。勝者が自動で次へ進む
                                              （スコアは省略可）

  終了後
    /tourney-export                        … 結果一覧をテキストファイルで出力


-----------------------------------------------------
■ 3. 困ったとき用のコマンド
-----------------------------------------------------
    /tourney-undo            … 直前の操作（結果入力・棄権・対戦表生成）を取り消し
    /tourney-dq 参加者名     … 棄権処理。進行中なら相手が不戦勝で進出
    /tourney-status          … 今の状態・対戦表・次の操作を確認
    /tourney-reset           … 大会データを完全リセット（確認ボタンあり）


-----------------------------------------------------
■ 4. 参加者に案内すること
-----------------------------------------------------
参加者に伝えるのはこの3つだけでOK:
    ・エントリーボタンを押して登録
    ・/tourney-myresult     … 自分の戦績・次の試合を確認（本人だけに表示）
    ・対戦表Webページ（下記URL）


-----------------------------------------------------
■ 5. 対戦表 Webページ
-----------------------------------------------------
公開URL:
    https://tellurium587465.github.io/yeg-tourney-bot/

・参加者はこのURLを開くだけで最新のブラケットを確認できる（60秒ごと自動更新）
・結果入力のたびに自動でWebに反映される（数十秒のタイムラグあり）


-----------------------------------------------------
■ 6. スプレッドシート連携
-----------------------------------------------------
保存のたびに以下のGoogleスプレッドシートへ自動同期される:
    「YeGタイピング大会」
    - 参加者シート（エントリー名簿）
    - 対戦表シート（全試合の状況）
    - 運営ログシート（操作履歴）


-----------------------------------------------------
■ 7. リマインド機能
-----------------------------------------------------
進行中の試合の結果が10分以上入力されないと、
#moderator-only チャンネルに通知が届く。


-----------------------------------------------------
■ 8. 権限について
-----------------------------------------------------
・運営コマンドはサーバー管理者、または「運営」ロールの人だけが使える
・手伝いスタッフに使わせたい場合は、その人に「運営」ロールを付与する


-----------------------------------------------------
■ 9. トラブル時のチェック
-----------------------------------------------------
・コマンドが反応しない
    → Botのウィンドウが開いて「ログイン成功」しているか確認
    → 落ちていたら 1.の起動方法で再起動

・コマンドが二重に反応する
    → Botを二重起動している。一度すべて閉じて、1つだけ起動し直す

・Webページが更新されない
    → GitHubのワークフロー実行状況を確認:
       https://github.com/tellurium587465/yeg-tourney-bot/actions

・スプレッドシートに反映されない
    → credentials.json がフォルダにあるか、スプレッドシートにサービスアカウントが
       編集者として共有されているか確認


-----------------------------------------------------
■ 10. 重要ファイル（さわらない・消さない）
-----------------------------------------------------
    .env             … トークンや各種設定（絶対に他人に見せない / GitHubに上げない）
    credentials.json … Google認証鍵（同上）
    tournament.json  … 現在の大会データ
    backups/         … 保存ごとの自動バックアップ（最新50件）
